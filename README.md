# VaxView

VaxView is a web-based immunisation data management system developed using Python, SQLite, HTML, CSS, and JavaScript. The application provides an intuitive interface for managing, searching, and viewing vaccination records while demonstrating full-stack web development concepts and database integration.

## Overview

The project was designed to simplify the management of immunisation data through a user-friendly web application. It allows users to access vaccination information efficiently and demonstrates the use of backend development, database management, and responsive web design.

## Features

- Vaccination record management
- Search and filtering functionality
- Database integration with SQLite
- Responsive and user-friendly interface
- Data viewing and management dashboard
- Dynamic web pages using Python backend

## Technologies Used

- Python
- SQLite
- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/VaxView.git
```

2. Navigate to the project directory:

```bash
cd VaxView
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
[http://localhost:80]
```
